### 关键字
```text

备注：main不是关键字，但是不能用，是java程序的程序入口。
```
### 标识符
```text
java程序中自定义的词汇都是标识符，如包名、类名、变量名、方法名等。
标识符规则：
    组成：字母a-z A-Z，数字0-9，下划线 _，美元符 $。
    注意：不能以数字开头，不能为关键字，要求见名知意。
常用类型标识符定义原则：
    包名：全小写，如 com.maker.basic
    类名：大驼峰，如 HelloWorld
    变量：，如 isLine = true
    常量：全大写，单词间用下划线分隔，如 CONTENT_TYPE = "json"
    方法名：小驼峰，如 getName() setName()
```
### 数据类型
```text
程序是用来处理数据的逻辑，完整的程序 = 数据 + 处理数据的逻辑，java语言中的数据类型有两大部分：
基本数据类型（四类八种）：
    整数: byte short int long
    小数: float double
    字符: char
    布尔: boolean
引用数据类型：
    字符串
    数组
    类
    接口
```
### 常量
```text
常量就是固定不变的数据，根据java基本数据类型，分为下述四种：
整数类型常量
    十进制整数常量表示法：正常数字，如 0，-5，100 等
    二进制整数常量表示法：以0b(0B)开头，如0b1011, 0B101 等
    八进制整数常量表示法：以数字0开头，如01, 07，0721 等
    十六进制整数常量表示法：以0x(0X)开头，数字0-9及A-F组成，如0x23A2 等
浮点数类型常量
    如1.0, -3.14, 5.812 等
布尔类型常量
    只有两个值：true, false
字符类型常量
    以英文单引号包裹的一个字符，只能是一个字符，不能多不能少，如 'A', '家', '#' 等
字符串类型常量
    以英文双引号包裹的任意个数字符，0-N个字符，如 "中国", "UTF-8" 等
null型常量
    null
```
